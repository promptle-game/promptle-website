Description

Promptle is a web game similar to Wordle but with AI implemented into it. This project addresses the limitation that Wordle games have, these are hard coded topics that can come from a hard set database, we address the problem by the implementation of AI through OpenAI API to dynamically create puzzles based on the users’ prompt they give. Our project will have a login system with profiles, “most common” wordle games, AI prompt generated game, multiplayer, and variety of other games, The purpose of this document is to explain the plans for our project after revision and counseling from instructors and compare changes that were made since then.
